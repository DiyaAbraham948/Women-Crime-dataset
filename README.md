Women Crime Analysis India

Project Overview
Examining crime against women in India (2001-2022) using NCRB data.

Data Source
This dataset was obtained from Mendeley Data.

- Dataset Name: Crime Against Women Dataset for Dark Spot identification in India
- Source Link:  https://data.mendeley.com/datasets/whrdh8c5zb/1/files/f63fe570-5fe5-4efe-a229-73dbc83205a7 
- Original Publisher: National Crime Records Bureau (NCRB) & Census of India
- Description: 21 years of NCRB crime statistics and census parameters across districts.

Dataset Contents
- `Dataset.csv` - Raw uncleaned data

Methodology
1. Data cleaning in Excel
2. Analysis and visualization in Power BI

Limitation 
Data represents only crimes reported to police.
